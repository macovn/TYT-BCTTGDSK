# Sổ Theo Dõi Truyền Thông GDSK – A11/TYT

Hệ thống quản lý hoạt động truyền thông giáo dục sức khỏe tại Trạm Y tế xã/phường theo mẫu A11/TYT.

## 🚀 Tính năng chính
1. **Quản lý dữ liệu**: Nhập liệu nhanh chóng, tự động đánh số thứ tự.
2. **Báo cáo tự động**: Thống kê số buổi, số người tham dự theo tháng.
3. **Xuất file chuẩn**: Xuất Excel (core), Word (in ấn) và PDF.

## 📋 Quy trình sử dụng
1. **Nhập liệu**: Điền thông tin buổi truyền thông vào bảng (Thời gian, Nội dung, Hình thức...).
2. **Phân loại**: Sử dụng các danh mục có sẵn (Hình thức, Đối tượng) để đảm bảo tính nhất quán.
3. **Kiểm tra**: Xem báo cáo tháng để nắm bắt tiến độ công việc.
4. **Lưu trữ**: Xuất file Excel hàng tháng để lưu trữ trên máy tính.
5. **In ấn**: Xuất file Word/PDF để in sổ theo dõi định kỳ.

## 🛠 Cấu trúc thư mục
- `/src`: Mã nguồn ứng dụng.
- `/docs`: Các mẫu tài liệu và hướng dẫn.
- `/jobs`: Các tác vụ xử lý dữ liệu.

## 💡 Gợi ý mở rộng
- Tích hợp chữ ký số cho báo cáo.
- Đồng bộ dữ liệu lên đám mây (Cloud Sync).
- Quản lý kho tài liệu truyền thông (Tờ rơi, Video).
